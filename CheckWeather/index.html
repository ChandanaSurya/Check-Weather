Description:

  • This site helps the users to know find a location and also indicates the current weather in the location that they desire to know.

Instructions to use our application:

  • Enter the city name you want to know about weather in that location.
  • Click on enter button.
  • It displays weather and temperature of that location.

Features:

  • It takes input as the users required location.
  • Weather API is used to know the weather of the location entered by user.
  • Google Maps API is used to return the location that user enters.
  • A logfile is also created and it is hidden from the user.

Software requirements:

  • Languages: HTML, CSS, JavaScript
  • Text Editor: Atom
  • Library: Bootstrap
